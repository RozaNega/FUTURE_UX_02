# FUTURE_UX_02

Suisse Clinic Mobile App

The Book Appointment feature allows users to easily schedule services through a simple and guided flow. Users select a service, choose a specialist, pick a date and time, and confirm their booking. The design focuses on clarity, minimal steps, and mobile-friendly interaction to ensure a smooth and efficient user experience.
